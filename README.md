# douban_pc_web
仿豆瓣



### 开发模式启动（客户端渲染）
```
npm run dev:client
```

### 开发模式启动（服务端渲染）
```
npm run dev:ssr
```

### 生产模式打包
```
npm run build
```

### 生产模式预览



```
npm run serve
```
